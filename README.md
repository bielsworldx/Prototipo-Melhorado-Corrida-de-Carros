
# Protótipo Melhorado Jogo de Corrida

Protótipo simples de um jogo de corrida multiplayer.


## Aprendizados

- Utilização de Matrizes
- Colisores


## Funcionalidades

- Movimentos com teclas direcionais e WASD
- Telas de derrota e vitória


## Stack utilizada

**Front-end:** JavaScript, CSS e HTMML



## Deploy

1. Clone ou baixe este repositório.

```bash
  git clone https://github.com/bielsworldx/Prototipo-Melhorado-Corrida-de-Carros
```
2. Abra o arquivo `index.html` em seu navegador.
